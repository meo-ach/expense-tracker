# 📄 TECH STACK (Simple Version)

## 💻 Frontend (What user sees)

This is the part of the website the user interacts with.

* **HTML** → Structure of the website
* **CSS** → Design and styling
* **JavaScript** → Makes the website interactive
* **React.js (optional but recommended)** → For building a fast and dynamic UI

---

## 🔙 Backend (Behind the scenes)

This handles logic and data.

* **Node.js** → Runs the server
* **Express.js** → Helps create APIs easily

---

## 🗄️ Database (Where data is stored)

Stores user data like expenses and income.

* **MongoDB** → Easy and beginner-friendly
  *(or MySQL if you prefer tables)*

---

## 🔐 Authentication (Login security)

Keeps user accounts safe.

* **JWT (JSON Web Token)** → For login sessions
* **bcrypt** → To encrypt passwords

---

## 📊 Charts / Insights

Used to show graphs and insights.

* **Chart.js** → Simple charts
  *(or Recharts if using React)*

---

## ☁️ Deployment (Making website live)

* **Frontend** → Vercel / Netlify
* **Backend** → Render / Railway
* **Database** → MongoDB Atlas

---

## 🛠️ Tools

* **VS Code** → Code editor
* **Git & GitHub** → Save and manage code
* **Postman** → Test APIs

---

## ✅ Simple Summary

* Frontend → HTML, CSS, JS, React
* Backend → Node.js, Express
* Database → MongoDB
* Charts → Chart.js
* Deployment → Vercel + Render
